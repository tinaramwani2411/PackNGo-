# PackNGo - Travel Website

PackNGo is a modern, responsive travel website designed to help users discover amazing destinations and book personalized travel packages. Built with HTML, CSS, and JavaScript, it offers a seamless user experience across all devices.

## 🌟 Features

### Core Functionality
- **5 Complete Pages**: Home, About Us, Destinations, Packages, Contact Us
- **Fully Responsive Design**: Optimized for mobile, tablet, and desktop
- **Modern UI/UX**: Clean design with smooth animations and interactions
- **Navigation & Footer**: Consistent navigation bar and comprehensive footer on all pages

### Travel Features
- **Destination Search & Filter**: Search destinations by name and filter by category (Beach, Mountain, City, Adventure)
- **Package Listings**: Detailed travel packages with pricing, duration, and target audience
- **Image Gallery**: Interactive gallery with hover effects showcasing destinations
- **Testimonials Section**: Real customer reviews and experiences
- **Contact Form**: Comprehensive contact form with validation

### Technical Features
- **No Backend Required**: Pure frontend implementation
- **Form Validation**: Client-side validation with user-friendly error messages
- **Smooth Scrolling**: Enhanced navigation experience
- **Interactive Elements**: Hover effects, button animations, and dynamic filtering

## 📁 Project Structure

```
PackNGo/
├── index.html          # Home page with hero, features, gallery, testimonials
├── about.html          # About us page with company story and stats
├── destinations.html   # Destinations page with search and filter functionality
├── packages.html       # Travel packages with pricing details
├── contact.html        # Contact page with form and company information
├── style.css           # Main stylesheet with responsive design
├── script.js           # JavaScript for interactivity and form validation
└── README.md           # Project documentation
```
